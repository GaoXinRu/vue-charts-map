# vuechartsmap

## VUE + Echarts + D3 绘制图表

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

版本对应: 

"d3": "^6.5.0"
"echarts": "^5.0.1"
"echarts-wordcloud": "^2.0.0"
"vue": "^2.6.11"
